# Fraud Detection Using Data Analysis

## Project Description
This project uses exploratory data analysis (EDA) to identify and analyze fraudulent transactions from a simulated dataset.

## Dataset Information
- **TransactionID**: Unique identifier for transactions.
- **TransactionAmount**: Amount of the transaction in USD.
- **FraudReported**: Binary variable (1 = Fraud, 0 = Not Fraud).

## Steps Performed
1. Loaded the dataset and performed basic exploratory analysis.
2. Visualized transaction distributions, fraud cases by type, and correlations.
3. Derived actionable insights to understand fraudulent behaviors.
